Quản trị Chấm Công

Cấu hình và tạo file index.html cho hệ thống chấm công

🔗 Kết nối Apps Script
Lấy từ Apps Script → Deploy → Manage deployments → Web app URL
🖼️ Logo công ty
Logo sẽ được nhúng thẳng vào file HTML (không cần host ảnh ngoài)
📷 Chưa có logo — upload ảnh để xem trước
🏢 Thông tin công ty
📍 Vị trí văn phòng
Lấy tọa độ: mở Google Maps → bấm chuột phải vào vị trí → copy tọa độ
🗺️ Chọn trên bản đồ
📏 Giới hạn khoảng cách
0 = không giới hạn. Ví dụ: 10000 = 10 km
✅ Trong 200m → Hợp lệ
⚠️ 200m – 10km → Cảnh báo
🚫 Hơn 10km → Chặn
💬 Tên ứng dụng
Ví dụ: "HSTC Chấm công", "Siêu Thanh Chấm công"
📄 Tạo file index.html
👁️ Xem trước cấu hình
Cấu hình sẽ được nhúng vào index.html
📋 Hướng dẫn sử dụng
Bước 1: Điền đầy đủ thông tin ở trên
Bước 2: Bấm "Tạo & Tải file" → tải về index.html
Bước 3: Upload file lên GitHub repository (thay file cũ)
Bước 4: Trang chấm công tự cập nhật sau ~1 phút

Lưu ý: File admin.html này chỉ dùng nội bộ để tạo cấu hình, không cần upload lên GitHub (hoặc upload nhưng không chia sẻ link cho nhân viên).
