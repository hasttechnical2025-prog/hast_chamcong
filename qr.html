📱 File cá nhân cho iPhone (iOS)

Ví dụ: https://hasttechnical2025-prog.github.io/hast_chamcong/nv/
Mỗi file sẽ có URL: .../nv/Ten_NhanVien.html
📋 Làm theo 3 bước:
📥 Bước 1 — Tải file về máy
☁️ Bước 2 — Upload lên GitHub
1. Vào github.com → repository → thư mục nv
2. Bấm Add file → Upload files
3. Kéo thả các file .html vào → Commit changes
📱 Bước 3 — QR code cho từng người
QR trỏ thẳng vào file cá nhân — quét → Add to Home Screen → ✅
📲 Hướng dẫn nhân viên iPhone
1. Quét QR code cá nhân
2. Safari mở → tên hiển thị đúng
3. Share ⎙ → Add to Home Screen
4. ✅ Tên luôn đúng, không bao giờ lỗi!

👥 Danh sách nhân viên

Nhập mỗi tên một dòng. Tên sẽ được dùng làm tham số trong link.
